# KWOL Registration

Fullstack приложение для регистрации в два шага

## Технологии

- **Frontend:** Next.js, React, TypeScript, Tailwind
- **Backend:** NestJS, Prisma, PostgreSQL
- **Infrastructure:** Docker, Docker Compose

## Репозиторий

https://github.com/kazantsev-developer/kwol-registration

## Запуск

```bash
git clone https://github.com/kazantsev-developer/kwol-registration.git
cd kwol-registration
docker-compose up -d
```
